# Blog

# About this project

This is a project to build a simple blog, allowing the users to control your posts and see another user's posts. This project is deployed at https://desafio-maino-isaac.herokuapp.com/

# Used Technologies

## Backend
- Ruby 
- Rails 
- Postgresql 


## Frontend

- Html 
- Css 
- Bootsrap 

# Requirements
- Ruby 2.7.1
- Ruby on Rails 6.1.4

# How to execute this project


```bash
# clone repository
git clone https://github.com/isaacbrds/desafio-maino.git 

# access the folder
cd desafio-maino

# Install gems

bundle install

# Create a database
rails db:create db:migrate 

# Install JS libs
yarn



```

# How to test this project

```bash 

# Run rspec 

rspec
```
