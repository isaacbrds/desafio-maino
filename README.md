# How to execute this project


```bash
# clone repository
git clone https://github.com/isaacbrds/desafio-maino.git 

# access the folder
cd desafio-maino

# Install gems

bundle install

# Create a database
rails db:create db:migrate 

# Install JS libs
yarn



```
